- Updated for iOS 26.
- New icon.
- New context menu: long press any favourite drink to edit it.
- Fixes issue where the default drinks list sometimes is not saved after installations; reinstall the app to bring it back.
- Fixes shortcuts.
- Fixes a very old bug which sets an incorrect FL OZ caffeine strength value for all default drinks on initial app install. 
- No longer erasing the original caffeine strength value after drink is updated with a different unit setting.
- Restored confirmation banner after logging a drink.

The update is [live on the App Store.](https://apps.apple.com/gb/app/caffeine/id1594448346)
